# Contributing to SmartChannels

Thank you for your interest in SmartChannels! We're not yet accepting external contributions as the project is in early development, but we will be soon.

## Getting Started

Once the project is open for contributions, this guide will cover:

- Setting up your local development environment
- Running tests
- Code style and conventions
- How to submit a pull request

## Code of Conduct

Be kind, be respectful, be constructive. We're building something to make AI assistants better for everyone.

## Questions?

Join the waitlist at [smartchannels.com](https://smartchannels.com) or open a GitHub Discussion.
